JQ AI Skills v0.3.5 is live.

This release improves the first-run path for the public skill library. Instead of asking visitors to inspect every skill folder first, it adds a practical selection guide, recommended starter sets, and clearer README onboarding for choosing the right first skill.

Use it when you want to start with one useful workflow:
