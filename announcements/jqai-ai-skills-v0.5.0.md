# JQ AI Skills v0.5.0 Is Live

Public-safe announcement copy for the JQ AI Skills v0.5.0 adoption layer release.

## Short Post

JQ AI Skills v0.5.0 is live.

This release adds the adoption layer for the public skills library:

- `ROADMAP.md` for the public direction of the repo,
- `SUPPORT.md` for help and issue guidance,
- GitHub issue templates for skill requests, install or bug reports, and docs feedback,
- a pull request template with validation and public-safety checks,
- and `docs/skill-quality-matrix.md` to compare maturity, examples, safety sensitivity, and install notes.

The repo is now easier to evaluate, request from, and contribute to.

Release:
https://github.com/jqaisystems/jqai-ai-skills/releases/tag/v0.5.0

Skill quality matrix:
https://github.com/jqaisystems/jqai-ai-skills/blob/main/docs/skill-quality-matrix.md
