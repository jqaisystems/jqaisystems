# JQ AI Skills v0.3.8 Is Live

Public-safe announcement copy for the JQ AI Skills v0.3.8 one-minute install release.

## Short Post

JQ AI Skills v0.3.8 is live.

This release adds a one-minute install guide for visitors who want to try one skill before reading the full catalog.

The guide shows how to:

- list available skills,
- install `github-safe-publisher`,
- reload the local AI coding tool,
- run a first prompt,
- check the copied skill folder,
- and follow safe first-run rules.
