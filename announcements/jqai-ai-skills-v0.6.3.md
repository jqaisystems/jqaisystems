# JQ AI Skills v0.6.3

JQ AI Skills v0.6.3 adds an update guide for installed skills.

The new guide shows how to refresh one installed skill or a reviewed set of skills after pulling the latest repo version.

It covers temporary-target checks, reinstall commands, reload steps, local edit handling, and copied-folder confirmation.
