# JQ AI Skills v0.8.1

JQ AI Skills v0.8.1 adds two proof-workflow skills for the part of a project that happens after the work ships.

The release adds:

- `github-profile-proof-updater`: refresh GitHub profile visuals, preserve README image paths, verify live GitHub rendering, and document proof trails.
- `social-campaign-proof-tracker`: plan LinkedIn/X proof campaigns, write platform-specific copy, add alt text, track published links, and prepare follow-up notes.

Why it matters:

Shipping useful work is only half of the public proof loop. The other half is making the work easy to inspect, verify, share, and revisit later. These two skills package that handoff into a repeatable workflow.
