# JQ AI Skills v0.3.3 Is Live

Public-safe announcement copy for the JQ AI Skills v0.3.3 release and GitHub safety review sample.

## LinkedIn

JQ AI Skills v0.3.3 is live.

This release makes the safety side of the library more concrete: it adds a fake-data public review sample for `github-safe-publisher`.

The sample shows what a review gate can look like before a public GitHub commit or push: artifact type, files inspected, scanner summary, verdict examples, manual checks, and named-file staging.

That is the pattern I want from reusable AI workflows: not just better prompts, but inspectable process, public-safe examples, and review discipline that someone else can understand before they use it.
