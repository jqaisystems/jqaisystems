# JQ AI Skills v0.4.5 Is Live

Public-safe announcement copy for the JQ AI Skills v0.4.5 security and responsible-use release.

## Short Post

JQ AI Skills v0.4.5 is live.

This release adds a root `SECURITY.md` file for security and responsible use.

It covers:

- what belongs in a sensitive report,
- what should not be posted in public issues,
- how to install skills safely,
- when to use `skill-reviewer`,
- how to keep private work out of public releases,
- and how `github-safe-publisher` and the release checklist fit into the publishing workflow.
