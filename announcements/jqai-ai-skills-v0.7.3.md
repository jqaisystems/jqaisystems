# JQ AI Skills v0.7.3

JQ AI Skills v0.7.3 adds a utility skill proof pack.

The new proof pack gives public-safe output shapes for `web-scraper`, `code-deduplicator`, and `em-dash-remover`, using fictional pages, code, and copy. It shows what these smaller utility skills are expected to produce before someone installs or adapts them.

- Utility skill proof pack: https://github.com/jqaisystems/jqai-ai-skills/blob/main/docs/examples/utility-skill-proof-pack.md
- Skill quality matrix: https://github.com/jqaisystems/jqai-ai-skills/blob/main/docs/skill-quality-matrix.md

Safety note: the proof pack uses fictional pages, code, and copy. It does not include private source, account data, exports, browser profiles, raw logs, credentials, customer records, local machine paths, or unpublished implementation notes.
