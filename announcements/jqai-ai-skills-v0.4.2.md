# JQ AI Skills v0.4.2 Is Live

Public-safe announcement copy for the JQ AI Skills v0.4.2 troubleshooting release.

## Short Post

JQ AI Skills v0.4.2 is live.

This release adds `TROUBLESHOOTING.md`, a root support page for common install and reload issues.

It covers:

- skills not appearing after install,
- wrong install targets,
- PowerShell and macOS/Linux shell issues,
- reinstalling or removing one skill safely,
- and when to use `skill-reviewer` before installing unknown skills.

The first-use path is now:

```text
START_HERE -> choose -> INSTALL -> troubleshoot if needed -> run -> understand output -> use bundles or catalog
```
