# JQ AI Skills v0.7.5

JQ AI Skills v0.7.5 adds a release-to-showcase handoff proof sample.

The new sample shows what should happen after a public GitHub release is published: update the profile, update the website source, build a small host upload package, verify the live URLs, and record the final project status.

Public links:

- Release: https://github.com/jqaisystems/jqai-ai-skills/releases/tag/v0.7.5
- Release showcase handoff sample: https://github.com/jqaisystems/jqai-ai-skills/blob/main/docs/examples/release-showcase-handoff-sample.md
- Skill quality matrix: https://github.com/jqaisystems/jqai-ai-skills/blob/main/docs/skill-quality-matrix.md
- Public proof index: https://github.com/jqaisystems/jqai-ai-skills/blob/main/docs/examples/public-proof-index.md
- Complete catalog: https://github.com/jqaisystems/jqai-ai-skills/blob/main/docs/catalog.md

Safety note: the handoff sample uses fictional release names, fictional URLs, and generic file paths. It does not include hosting credentials, account screens, admin URLs, analytics, deployment logs, local machine paths, raw prompts, private chat logs, client material, or unpublished operational notes.
