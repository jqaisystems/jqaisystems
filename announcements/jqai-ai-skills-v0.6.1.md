# JQ AI Skills v0.6.1

JQ AI Skills v0.6.1 adds a quick reference walkthrough for the public skill library.

The new walkthrough shows the route from cloning the repo to listing skills, testing a temporary install target, installing `github-safe-publisher`, reloading the tool, and running the first safe prompt.

It is designed for visitors who want to use the short command map without guessing the order of steps.
