# JQ AI Skills v0.6.5

JQ AI Skills v0.6.5 adds a skill library evaluation checklist.

The new checklist helps visitors decide whether the library is understandable, installable, versioned, public-safe, and ready for a test install.

It covers repo identity, license, install path, folder shape, safety boundaries, example quality, release discipline, first-skill choice, red flags, and a final scorecard.
