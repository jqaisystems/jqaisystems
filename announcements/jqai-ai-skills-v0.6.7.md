# JQ AI Skills v0.6.7

JQ AI Skills v0.6.7 adds a first skill walkthrough.

The walkthrough helps visitors move from choosing one safe first skill to running it with fake input before touching real work. It uses `github-safe-publisher` as the example and shows the score, temporary install, fake candidate files, first prompt, expected verdicts, and stop conditions.

Public links:

- Release: https://github.com/jqaisystems/jqai-ai-skills/releases/tag/v0.6.7
- First skill walkthrough: https://github.com/jqaisystems/jqai-ai-skills/blob/main/docs/examples/first-skill-walkthrough.md
- First skill scorecard: https://github.com/jqaisystems/jqai-ai-skills/blob/main/docs/guides/first-skill-scorecard.md
- First run sample: https://github.com/jqaisystems/jqai-ai-skills/blob/main/docs/examples/first-run-github-safe-publisher.md
- Public proof index: https://github.com/jqaisystems/jqai-ai-skills/blob/main/docs/examples/public-proof-index.md

Safety note: use fake or public-safe input for the first run. Do not test with credentials, account data, logs, exports, customer records, screenshots, unpublished prompts, private project notes, or local private paths.
