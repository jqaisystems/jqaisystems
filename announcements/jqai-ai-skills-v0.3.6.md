# JQ AI Skills v0.3.6 Is Live

Public-safe announcement copy for the JQ AI Skills v0.3.6 release, public catalog, and stable 2026 social preview.

## Short Post

JQ AI Skills v0.3.6 is live.

This is a public-proof polish release. It adds a complete public catalog for all 12 installable skills, links that catalog from the README and skill selection guide, and switches the social preview to a stable `updated 2026` banner that does not need to change on every patch release.

Use the catalog when you want the full view of the library:

- What each skill does
- Who it helps
- When to use it
- Recommended bundles
- Safe install defaults
