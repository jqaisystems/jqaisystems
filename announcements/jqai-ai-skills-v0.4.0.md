## Short Post

JQ AI Skills v0.4.0 is live.

This release adds `START_HERE.md`, a top-level onboarding path for visitors who want to understand the repo, choose one skill, install it, run a first prompt, compare the output, and then move into workflow bundles or the full catalog.

The release keeps the stable 2026 social preview unchanged and makes the public visitor path clearer:

```text
START_HERE -> choose -> install -> run -> understand output -> use bundles or catalog
```
