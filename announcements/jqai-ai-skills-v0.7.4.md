# JQ AI Skills v0.7.4

JQ AI Skills v0.7.4 adds a demo animation v2 walkthrough sample.

The new sample uses a fictional Signal Desk product brief to show what a finished responsive walkthrough should contain before someone uses `demo-animation-v2` on a real project. It covers the expected screen sequence, captions, timeline, desktop checks, mobile checks, and public-safety review points.
