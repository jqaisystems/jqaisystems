# JQ AI Skills v0.9.0

JQ AI Skills v0.9.0 adds nine new skills, growing the public library from 23 to 32. It is the largest release so far and introduces two new categories.

Client Work covers the freelance engagement end to end: a client workspace scaffolder that builds the project folder from a reusable template, a discovery brief that interviews through seven areas and writes the brief plus a strategy starter, a proposal writer that drafts from your own service tiers, and a follow-up radar that scans outreach files for overdue follow-ups and drafts them oldest first. Agent Workflow makes the AI-assisted work itself repeatable: a skill builder for authoring new skills correctly, a session handoff that freezes state into a start-here document, an AGENTS.md writer that serves one skill set to both Claude Code and Codex, and an ops status report with an Attention Needed section. A Windows screen recorder, previously website-only, joins the repo.

Safety note: every business skill collects user-specific values (pricing tiers, folder templates, watch folders, report configs) through ask-once configuration files created on first run. The drafting skills never send anything, the radar reports before it drafts, and the status report is read-only except for the report file. No personal data, credentials, client material, pricing, or private prompts ship inside the skill files, and the repo-level public safety scan reports READY.
