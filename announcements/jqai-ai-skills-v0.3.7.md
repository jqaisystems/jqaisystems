# JQ AI Skills v0.3.7 Is Live

Public-safe announcement copy for the JQ AI Skills v0.3.7 workflow bundles release.

## Short Post

JQ AI Skills v0.3.7 is live.

This release adds a dedicated workflow bundles example page. Instead of only showing each skill one by one, the repo now shows three practical multi-skill paths:

- Public GitHub publishing.
- Content and research.
- Demo and launch.

Use it when a job needs a small sequence of repeatable workflows, such as safety review, public proof, release copy, public source curation, or demo launch support.
