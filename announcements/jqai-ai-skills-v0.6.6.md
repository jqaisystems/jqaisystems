# JQ AI Skills v0.6.6

JQ AI Skills v0.6.6 adds a first-skill scorecard.

The scorecard helps visitors move from evaluating the library to choosing one safe, testable first install. It scores job match, safe first test, output clarity, install confidence, and reuse value.

Public links:

- Release: https://github.com/jqaisystems/jqai-ai-skills/releases/tag/v0.6.6
- First skill scorecard: https://github.com/jqaisystems/jqai-ai-skills/blob/main/docs/guides/first-skill-scorecard.md
- Evaluation checklist: https://github.com/jqaisystems/jqai-ai-skills/blob/main/docs/guides/evaluate-skill-library.md
- Public proof index: https://github.com/jqaisystems/jqai-ai-skills/blob/main/docs/examples/public-proof-index.md

Safety note: test one skill with fake or public-safe input first. Do not use credentials, account exports, logs, customer data, private project notes, or local private paths as first-run material.
