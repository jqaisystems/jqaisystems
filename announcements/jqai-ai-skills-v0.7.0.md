# JQ AI Skills v0.7.0

JQ AI Skills v0.7.0 adds a first install proof trail.

The first install proof gives visitors one end-to-end path from clone to expected review: choose one safe first skill, install it into a temporary target, verify the copied folder, run the fake candidate, and compare the result with the expected `github-safe-publisher` review shape.

Safety note: the proof trail uses a disposable target and fictional candidate files. Do not use account exports, raw logs, screenshots, unpublished notes, or real project files for the first proof run.
