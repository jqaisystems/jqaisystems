These assets support the GitHub profile README for JQ AI SYSTEMS.

The README uses stable filenames so images can be refreshed without changing the
public profile markup. The top hero is intentionally kept separate from the
workflow, tile, and snapshot refresh set.

## Public-Safe Boundaries

- No client names, account screenshots, private dashboards, local paths, credentials, logs, browser tabs, or raw prompts.
- Use fictional or generic interface labels only.
- Keep text short and legible; let the README carry the detailed copy.
- Do not replace the stable 2026 `jqai-ai-skills` social preview with these profile visuals.
